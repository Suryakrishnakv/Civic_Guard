\chapter{SUPPORTING LITERATURE}

\section{Laravel Documentation}
The primary source of technical literature for this project is the official Laravel documentation. Laravel's expressive syntax and built-in tools for authentication, database management, and routing provided the backbone for CivicGuard.

\section{Modern Web Design Patterns}
The project also draws inspiration from modern UX/UI design patterns, particularly the use of Tailwind CSS for responsive, utility-first styling. Research into community-driven platforms like FixMyStreet provided insights into the operational challenges of public asset reporting.
