\chapter{DEPLOYMENT}
\section{Local Server Deployment}
CivicGuard is currently deployed in a local development environment using XAMPP. The application is served using PHP's built-in development server.

\section{Folder Structure}
The project follows the standard Laravel directory structure:
\begin{itemize}
    \item \texttt{app/}: Core logic (Models, Controllers).
    \item \texttt{resources/views/}: UI templates (Blade).
    \item \texttt{public/}: Publicly accessible assets.
    \item \texttt{storage/}: User-uploaded content (Reports/Resolution Photos).
\end{itemize}
