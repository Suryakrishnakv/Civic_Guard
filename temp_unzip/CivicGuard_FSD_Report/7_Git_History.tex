\chapter{GIT HISTORY}
To ensure proper version management, all developments of CivicGuard are tracked using Git. This allows for efficient debugging, feature branching, and collaborative development.

\begin{itemize}
    \item \textbf{Initial Commit}: Project boilerplate and base models.
    \item \textbf{Feature/Auth}: Completed multi-role login system.
    \item \textbf{Feature/Reporting}: Implemented image uploads and routing logic.
    \item \textbf{Feature/Admin}: Added department-level dashboards.
\end{itemize}

The repository serves as a central hub for all project assets, ensuring the source code is safe and versioned.
